# Stratio Meta #

## Start the Server ##

...
  > ./meta-server
...

## Start/stop/restart the Server as a daemon ##
jsvc needs to be installed ( http://commons.apache.org/proper/commons-daemon/jsvc.html )

...
  > ./meta-server-daemon start
  > ./meta-server-daemon stop
  > ./meta-server-daemon restart
...



## Start the Client ##

...
  > ./meta-client
...
